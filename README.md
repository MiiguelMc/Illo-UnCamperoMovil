# Illo un Campero - App Móvil 📱

Este repositorio contiene la aplicación móvil del proyecto **Illo un Campero**, desarrollada de forma nativa para Android utilizando **Kotlin**. La app está diseñada para que los clientes puedan realizar sus pedidos de camperos de forma rápida e intuitiva.

## 🚀 Tecnologías
* **Lenguaje:** [Kotlin](https://kotlinlang.org/)
* **Entorno:** Android Studio
* **Arquitectura:** MVVM (Model-View-ViewModel)
* **Diseño:** Jetpack Compose (o XML)

## 📋 Requisitos previos
* Android Studio (Versión reciente)
* JDK 17 o superior
* Dispositivo Android o Emulador (mínimo API 24)
